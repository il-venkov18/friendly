## Установка

Инструкции по установке и запуску проекта:

1.  Клонируйте репозиторий:

    ```bash
    git clone [URL вашего репозитория]
    ```

2.  Переключитесь на ветку разработки фронтенда:

    ```bash
    git fetch origin
    git checkout front-dev
    ```

3.  Перейдите в директорию проекта:

    ```bash
    cd .\frontend\
    ```

4.  Установите зависимости:

    ```bash
    npm install
    ```

5.  Запустите проект:

    ```bash
    npm run dev
    ```

6.  Откройте страницу в браузере:

    Доступ по ссылке: https://localhost:5173/

7.  Перейдите в адаптивный режим:

    ```bash
    Ctrl + Shift + M
    ```

8. Выберите размер соответствующий дизайну в фигме (390 х 844):

    ```bash
    Режим iPhone 12/13 + Рю i0S 14.6
    ```
