## Установка

Инструкции по установке и запуску проекта:

1.  Клонируйте репозиторий:

    ```bash
    git clone [URL вашего репозитория]
    ```

2.  Перейдите в директорию проекта:

    ```bash
    cd .\frontend\
    ```

3.  Установите зависимости:

    ```bash
    npm install
    ```

4.  Запустите проект:

    ```bash
    npm run dev
    ```

5.  Откройте страницу в браузере:

    Доступ по ссылке: https://localhost:5173/

6.  Перейдите в адаптивный режим:

    ```bash
    Ctrl + Shift + M
    ```

7. Выберите размер соответствующий дизайну в фигме (390 х 844):

    ```bash
    Режим iPhone 12/13 + Рю i0S 14.6
    ```
